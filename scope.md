# Lowcountry Guns & Range — Scope of Work

## Current Sprint: Marketing Site Rebuild

### In Scope
- Full rebuild of public-facing marketing site (Next.js App Router)
- All page content migration from legacy WordPress site (lcguns.com)
- Memberships, Training, Range Info, Pro Shop, Rentals, IDPA, Private Events
- Image optimization and legacy media migration
- SEO metadata and structured data
- Responsive design (mobile, tablet, desktop)
- Google Analytics 4 / GTM integration
- Vercel deployment and CI/CD via GitHub

### Out of Scope (Unless Explicitly Requested)
- E-commerce / online store functionality
- Booking/reservation system
- Custom CMS or admin panel beyond existing integrations
- Native mobile app
- Email marketing platform build-out
